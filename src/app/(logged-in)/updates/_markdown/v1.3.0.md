## v1.3.0 - Layout revisado, status e links externos para perfis e mais

Oi, gente! Sejam bem-vindos ao **primeiro anúncio de atualização** dentro do MPost!
Dessa vez já temos umas novidades, então vamos explicar cada uma, uma por uma:

### 1. Status e Links externos para perfis

![Exemplo de perfil com Status e Link.](https://files.catbox.moe/3ifk5g.png)

Agora, indo para **Opções > Perfil**, você pode alterar as colunas de
**Status** e de **Link Externo**. Os mesmos irão aparecer na sua página de perfil
assim que você definir os mesmos.

### 2. Fixar postagens

![Exemplo de perfil com postagem fixada.](https://files.catbox.moe/msqpn6.png)

Agora você pode deixar uma mensagem fixada no seu perfil!
Vale ressaltar - só se pode ter uma mensagem fixada por vez. Então escolha bem!
Mostre pro mundo o que você é.

### 3. Layout levemente alterado

O MPost sempre foi visualmente bom, mas sempre tinha algo de errado aqui e ali.
Então, essas inconsistênciaszinhas foram consertadas e certos elementos agora
estão em posições mais coerentes a fim de melhorar a UX. Dá pra reparar olhando
na barra de navegação no topo da tela, que antigamente era uma barra de navegação
lateral.
