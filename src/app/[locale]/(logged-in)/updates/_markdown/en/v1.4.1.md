## Update v1.4.1 - Hotfix

Hey!! A few things were still a bit unfinished on the last update.
So here's a bit of a polish translation-wise, as well as bugfixes
for issues my friends notified me about (thank you [yuka](https://mpost.maciel.app/en/users/dls)
and [laura](https://mpost.maciel.app/en/users/leaf) for helping me find out about
these!)

Missing translations were correctly treated:

- User editing success toast text
- "Next" and "Previous" buttons on the home page/user page
- Edit post validation messages
- Global error page

Not only that, but a few bugfixes too. Here's a list of all the changes that were made:

- Fixed post validation issues (using spaces/enter would cause weird validation issues)
- Fixed posts now correctly being marked as "edited" if they have been edited
- Fixed password input autocomplete issues
- Fixed profile picture submission file validation (it used to verify the file size first, then the file type)
- Comments have a 140 character limit now (they had none)

That's it for today! I got more to show you later :))
See ya!
