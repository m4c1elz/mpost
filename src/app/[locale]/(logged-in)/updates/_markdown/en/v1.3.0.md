## v1.3.0 - Revised layout, status and external URLs for profiles, and more

Hey, guys! Welcome to the first update announcement inside MPost!
This time we got some new things already. Few, but we have 'em!

### 1. Status and external URLs for profiles

![Profile example with status and URL.](https://files.catbox.moe/3ifk5g.png)

Now, inside **Settings > User** you can modify the **Status** and the
**External URL** fields. These will appear on your profile page once you
set these.

### 2. Pin posts

![Profile example with a pinned post.](https://files.catbox.moe/msqpn6.png)

You can now leave a pinned message on your profile! Worth noting - you can only
have one message pinned at a time. So choose wisely!
Show the world what you are.

### 3. Slightly altered layout

MPost has always been visually good, but there was always something
wrong here and there. So, these little inconsistencies have been fixed
and certain elements are now in more coherent positions to improve UX.
You can notice by looking in the navigation bar at the top of the screen,
which used to be a sidebar.

Oh, and we have a footer now. Cool.

Well — those were the main changes. Sorry if I'm not releasing too many features,
I'm fixing several parts of the code that I _thought_ that I had done well writing,
but in fact it was only a stress for my self current 💔

I haven't stopped here yet - I have a lot of functionality ideas yet. I wrote it down
every suggestion that was given and I will still work on them! I really appreciate
everyone who is using MPost, it means a lot to me!! To such a simple little project,
I was so excited by the reactions of others. I love to see my little friends posting here!!

That's it for today. See you guys soon! It's not over yet...

**- Maciel, MPost dev**
